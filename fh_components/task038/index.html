分数表

表格排序组件说明：

  1. 用户需要提供一个数组，例如： 
               var klass =  [{
                "姓名": "小红",
                "数学": 89,
                "语文": 99,
                "英语": 77,
                "总分": 265
            }, {
                "姓名": "阿文",
                "数学": 99,
                "语文": 90,
                "英语": 33,
                "总分": 222
            }]
  2. 用户需要手动创建一个 <table> 并且设置一个 id为任意字符串，如： <table id="fh-table"></table>
  3. 调用函数：传入相应的值，如createTable("fh-table",klass)
  4. 只需要上述的步骤即可完成排序功能，若需要应用组件默认样式，则还需要设置 className 为 fh-table
  5. 默认样式中只对最外层设置了边框，若想每一项都有边框，则还需要在 <table> 加上 border="1"
